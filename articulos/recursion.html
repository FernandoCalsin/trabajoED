Publicación 2: Recursión

La recursión es una técnica matemática y computacional donde un problema se define en términos de sí mismo. Se basa en dos elementos clave:

Caso Base

Es la condición que detiene la recursión. Representa la solución más simple del problema y evita que el proceso se repita indefinidamente.

Regla Recursiva

Define cómo el problema se divide en subproblemas más pequeños, acercándose progresivamente al caso base.

De forma general, una definición recursiva puede expresarse así:

\[ f(n) = \begin{cases} \text{caso base}, & \text{si } n = n_0 \\ f(n - 1) + \text{operación}, & \text{si } n > n_0 \end{cases} \]

Esta forma indica que el valor de una función depende de su propio resultado para un valor menor, hasta alcanzar el caso base.

Un ejemplo clásico es el factorial:

\[ n! = \begin{cases} 1, & \text{si } n = 0 \\ n \times (n - 1)!, & \text{si } n > 0 \end{cases} \]

Aplicaciones

  • Programación: facilita la solución de problemas complejos dividiéndolos en subproblemas.
  • Estructuras de datos: árboles y grafos se definen naturalmente de forma recursiva.
  • Análisis de algoritmos: permite calcular tiempos de ejecución mediante relaciones de recurrencia.
