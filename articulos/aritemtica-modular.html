Matemáticas discretas

Publicación 4: Aritmética Modular

La aritmética modular es una rama de las matemáticas que estudia los números a partir del residuo que resulta al dividir un número entre otro. Se utiliza cuando los valores “vuelven a empezar” después de alcanzar un cierto límite, como ocurre en los relojes, los calendarios o los sistemas digitales.

Se expresa mediante la congruencia:

\[ a \equiv b \pmod{n} \]

Esto significa que los números \(a\) y \(b\) tienen el mismo residuo al dividirse entre \(n\).

Congruencia Modular

Dos números son congruentes módulo n si su diferencia es divisible entre n. En otras palabras, pertenecen a la misma clase de residuos.

\[ a \equiv b \pmod{n} \iff n \mid (a - b) \]

Suma Modular

En la suma modular, el residuo de la suma depende únicamente de los residuos de los números que se están sumando.

\[ (a + b) \bmod n = ((a \bmod n) + (b \bmod n)) \bmod n \]

Resta Modular

La resta modular funciona de manera similar a la suma, manteniendo siempre el resultado dentro del conjunto de residuos posibles.

\[ (a - b) \bmod n = ((a \bmod n) - (b \bmod n)) \bmod n \]

Multiplicación Modular

En la multiplicación modular, se multiplican los residuos y luego se toma nuevamente el módulo.

\[ (a \cdot b) \bmod n = ((a \bmod n) \cdot (b \bmod n)) \bmod n \]

Aplicaciones

  • Criptografía: base de sistemas de cifrado y firmas digitales.
  • Relojes digitales: controlan ciclos repetitivos.
  • Sistemas de seguridad informática: protege contraseñas y comunicaciones.
  • Informática: se usa en funciones hash y control de errores.
